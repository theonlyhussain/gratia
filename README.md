# Gratia

Gratia is a privacy-first, premium local music player for Android with a beautiful UI and a local-first architecture.

---

## Download

Download the latest APK from the Releases page.

[GitHub Releases](https://github.com/theonlyhussain/gratia/releases)

---

## Features

* Local music playback
* Media3 / ExoPlayer
* Beautiful redesigned UI
* Floating glass mini player
* Album browsing
* Artist browsing
* Folder browsing
* Playlist support
* Favorites
* Search
* Lyrics
* Dynamic Light/Dark theme
* Room database
* Privacy-first
* Offline playback

## Privacy

Gratia:
* does not upload user music
* does not require an account
* stores everything locally
* does not track users

---

## Building

```bash
git clone https://github.com/theonlyhussain/gratia.git
cd gratia
./gradlew clean assembleDebug
```
